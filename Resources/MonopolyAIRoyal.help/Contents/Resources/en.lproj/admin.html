MonopolyAIRoyal Help

Admin & Settings

The Admin screen provides comprehensive control over all aspects of MonopolyAIRoyal. It features a left sidebar with 8 configuration sections and a content area on the right.

Saving Settings

Click Save All (green button at the bottom of the Admin sidebar) to persist all changes to the database and Keychain.

Configuration Sections

1. Models

Browse and search the full catalog of 200+ AI models available through OpenRouter:

  • Search by model name or ID
  • View context length, input cost, and output cost per 1M tokens
  • Models are fetched from the OpenRouter API (falls back to built-in list if offline)

Use this section to discover models for creating custom AI players.

2. Rules

Configure core game rules and house rules. See Game Rules for full details.

  • Starting cash, Go salary, jail bail, max turns
  • Win condition selection
  • House rule toggles (Free Parking Jackpot, Auction, etc.)

3. Tournament

Set up tournament formats for multi-game series:

  • Format: Single, Best of 3, Best of 5, Round Robin, Swiss
  • Series length: Number of games in a series
  • Schedule: View and manage scheduled games

4. Streaming

Configure live streaming output. See Streaming Setup for details.

5. Narrator

Configure text-to-speech narration. See Narration & Voice for details.

6. Budget

Set cost limits and monitor API spending. See Cost Tracking & Budgets for details.

7. API Keys

Manage API keys for OpenRouter and ElevenLabs. See API Keys for details.

8. Scheduling

Set up automatic game scheduling:

  • Auto-start: Toggle automatic game launching
  • Scheduled time: Set the time for auto-start (HH:MM)
  • Repeat: Never, Daily, or Weekly
  • Unattended mode: Run games without user interaction
Combine scheduling with budgets to run overnight tournaments with cost caps automatically enforced.
