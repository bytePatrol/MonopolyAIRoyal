Cost Tracking & Budgets

MonopolyAIRoyal includes comprehensive cost tracking and budget management to keep your API spending under control.

How Costs Work

Each time an AI player makes a decision in Live Mode, the app sends a request to OpenRouter. Costs are calculated as:

Cost = (input_tokens / 1,000,000) x input_rate + (output_tokens / 1,000,000) x output_rate

Budget Limits

Configure limits in Admin > Budget:

LimitDefaultPurpose
Per Decision$0.05Maximum cost for a single AI decision
Per Game$2.00Maximum total cost for one complete game
Per Day$10.00Maximum spending in a calendar day
Per Month$50.00Maximum spending in a calendar month

What Happens at the Limit

When any budget limit is reached, the app automatically falls back to Mock Mode for remaining decisions. The game continues with scripted AI behavior instead of real API calls.

Cost Monitoring

Sidebar Ticker

The bottom of the navigation sidebar shows your all-time total API cost, updating in real-time as decisions are made.

Budget Dashboard

In Admin > Budget, you can see:

  • Total all-time cost
  • Today's cost with percentage of daily limit (progress bar)
  • This month's cost with percentage of monthly limit
  • Current game cost with percentage of game limit

Post-Game Cost

The Post-Game screen includes the game's total API cost in the thumbnail subtitle.

Cost-Saving Tips

  • Use Gemini Flash or DeepSeek — they cost 10-100x less than GPT-4
  • Run at Fast or Instant speed — templates replace some AI commentary calls
  • Use Mock Mode for testing and casual play
  • Set conservative budget limits and monitor the dashboard
  • Use multiple fallback API keys to avoid rate limits (which can cause retries)
