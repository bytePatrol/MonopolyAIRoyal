The Game Screen

The Game screen is a 3-panel layout that shows the live game state, AI reasoning, and commentary in real-time.

Layout Overview

Left Panel — Player Status (240px)

Shows all players sorted by net worth (highest first):

  • Player name with color indicator
  • Current cash in hand
  • Net worth (cash + properties + buildings)
  • Net worth bar — visual percentage relative to $6,000
  • Property count owned
  • House/hotel count
  • Status badge — Ready, Thinking, Waiting, or Bankrupt

Center — Monopoly Board

The full 40-space Monopoly board displayed as an 11x11 grid:

  • Player tokens appear as colored circles with initials
  • Active player token pulses with animation
  • Properties show their color group, owner, and building count
  • Center area displays the latest dice roll and current event
  • Tokens animate smoothly between spaces

Right Panel (320px) — Two Sections

AI Brain Panel (Top)

The Brain Panel shows the active player's AI reasoning in real-time:

  • Header: Active player name with pulsing status dot
  • Reasoning area: Streaming text of the AI's thought process as tokens arrive
  • Footer stats:
    • Confidence percentage (0–100%)
    • Decision cost in USD
    • Model name being used
In Live Mode, you can read the AI's actual strategic reasoning. In Mock Mode, scripted reasoning is shown.

Commentary Feed (Bottom)

Live ESPN-style play-by-play commentary:

  • Events are shown newest-first with timestamps
  • Each entry has an emoji icon and color-coded type badge:
    • Buy — Property purchases
    • Trade — Player trades
    • Rent — Rent payments
    • Bankrupt — Player elimination
    • Build — House/hotel construction
  • Maximum 50 entries displayed (oldest removed)

Top Control Bar

Left Side

  • Turn counter: "TURN [N]" with active player count
  • Phase badge: Shows current game phase with color-coded pulsing dot

Right Side — Controls

Speed Control

Click the speedometer dropdown to adjust game speed:

SpeedTurn DelayPhase Delay
Slow5.0 seconds2.5 seconds
Normal2.5 seconds1.2 seconds
Fast0.8 seconds0.3 seconds
Instant0 seconds0 seconds
At Fast and Instant speeds, AI commentary falls back to pre-written templates instead of waiting for AI-generated narration.

Pause / Resume

Toggle the game between paused and running. When paused, the game loop stops and all processing halts until you resume.

End Game

Click End Game (red button) to immediately stop the game. The winner is determined by highest net worth at that moment. The app transitions to the Post-Game screen.
