MonopolyAIRoyal Help

Getting Started

System Requirements

  • macOS 15.0 (Sequoia) or later
  • Internet connection (for AI-powered games)
  • OpenRouter API key (free tier available)

First Launch

When you first open MonopolyAIRoyal, the app creates a local database at:

~/Library/Application Support/MonopolyAIRoyal/monopoly.sqlite

This stores your game history, ELO rankings, settings, and API usage records.

Setting Up Your API Key

  1. Navigate to Admin (gear icon in the sidebar)
  2. Select the API Keys section
  3. Enter your OpenRouter API key in the Primary Key field
  4. Click Validate to confirm it works
  5. Click Save All at the bottom of the sidebar
Get a free OpenRouter API key at openrouter.ai. Many models offer free tiers.

Running Your First Game

  1. Go to the Lobby — Click the game controller icon in the sidebar
  2. Select Players — Click on 2–6 AI player cards to select them. Selected players show a highlighted border
  3. Generate Trash Talk (optional) — Click the button to have AIs banter before the game
  4. Start the Game — Click the Start button. A 5-second countdown begins
  5. Watch the Action — The game auto-plays. AI players roll dice, buy properties, pay rent, and make strategic decisions
  6. Review Results — When the game ends, the Post-Game screen shows the winner, highlights, and standings

Mock Mode vs Live Mode

Mock Mode (No API Key)

Without an API key, the app runs in Mock Mode. AI decisions are scripted based on each player's personality type. This is great for testing and exploring the app without incurring any API costs.

Live Mode (With API Key)

With a valid API key, each AI player sends game state to their assigned LLM and receives real strategic decisions. You can watch the AI's reasoning stream in real-time in the Brain Panel.
