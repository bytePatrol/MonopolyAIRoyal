Game Rules & Mechanics

MonopolyAIRoyal implements the full Monopoly rule set with configurable house rules and win conditions.

Win Conditions

Last Standing (Default)

The last player who hasn't gone bankrupt wins. All other players must be eliminated.

Highest Net Worth at Turn Limit

When the max turn limit is reached, the player with the highest total net worth (cash + property values + buildings) wins.

First to $5,000

The first player to reach a net worth of $5,000 wins immediately.

Rent

  • Base rent: Listed on the property deed
  • Monopoly bonus: If a player owns all properties in a color group with no buildings, base rent is doubled
  • With houses: Rent increases progressively with 1–4 houses
  • With hotel: Maximum rent (equivalent to 5 houses)
  • Mortgaged: No rent collected on mortgaged properties

Building

  • You must own all properties in a color group (monopoly) to build
  • Maximum 4 houses per property, then upgrade to a hotel
  • Even building rule: You cannot build more houses on one property than another in the same group
  • Hotels replace all 4 houses and cost one additional house price

Mortgaging

  • Mortgage value = half the purchase price
  • Unmortgage cost = mortgage value + 10% interest
  • All buildings must be sold before mortgaging
  • Mortgaged properties cannot collect rent

Bankruptcy

A player is bankrupt when they cannot cover a debt even after liquidating all assets:

  1. Sell houses/hotels back to bank at 50% cost
  2. Mortgage remaining properties at 50% value
  3. If still unable to pay, the player is bankrupt
  4. All properties return to the bank (unowned)
