MonopolyAIRoyal Help

API Keys & Configuration

MonopolyAIRoyal uses API keys to connect to AI model providers and voice services. All keys are stored securely in your macOS Keychain.

OpenRouter API Keys

OpenRouter is the gateway to 200+ AI models. You need at least one key for Live Mode.

Key Slots

SlotPurpose
Primary KeyUsed for all API calls by default
Fallback Key 1Automatically used if primary is rate-limited
Fallback Key 2Third option in the rotation
Fallback Key 3Fourth option in the rotation

Key Rotation

If the current key receives a rate limit response (HTTP 429), the app automatically rotates to the next available key. If all keys are exhausted, the app falls back to Mock Mode for the remainder of the game.

Validating Keys

  1. Enter your key in the password field
  2. Click Validate
  3. The status indicator shows:
    • Valid — Key works, models can be fetched
    • Invalid — Key rejected by OpenRouter
    • Checking — Validation in progress
When a valid key is confirmed, Mock Mode is automatically disabled and the app switches to Live Mode.

ElevenLabs API Key

Required only if you want to use ElevenLabs voices for narration. Enter the key and click Validate to confirm.

Security

  • All keys are stored in the macOS Keychain — not in plain text files
  • Keys are never displayed in full after saving (password field masking)
  • Keys are retrieved securely on app launch

Getting API Keys

OpenRouter

Sign up at openrouter.ai and navigate to your API keys page. Many models have free tiers for low-volume usage.

ElevenLabs

Sign up at elevenlabs.io and get your API key from your profile settings. A free tier is available with limited characters per month.
