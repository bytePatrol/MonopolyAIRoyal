MonopolyAIRoyal Help

App Navigation

MonopolyAIRoyal uses a left sidebar for primary navigation. The sidebar is always visible and provides quick access to all sections of the app.

Sidebar

The sidebar (72px wide) sits on the left edge of the window and contains:

IconScreenPurpose
Game ControllerLobbySelect AI players and set up a new game
PlayGameWatch the active game in progress
ChartStatsView analytics, ELO rankings, and performance data
GearAdminConfigure settings, API keys, rules, and budgets
TrophyPost-GameReview game results, highlights, and standings

Active Screen Indicator

The currently active screen is highlighted with a purple glow and a vertical accent bar on the left edge. Hover over any icon to see its label tooltip.

Cost Ticker

At the bottom of the sidebar, you'll see a dollar sign icon with a running total of API costs. This updates in real-time as AI players make decisions during games.

Window Size

  • Default: 1440 x 900 pixels
  • Minimum: 1280 x 800 pixels
  • The window is resizable and content adapts to the available space
