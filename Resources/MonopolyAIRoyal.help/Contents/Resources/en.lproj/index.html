Quick Start

  1. Open the Admin tab and enter your OpenRouter API key
  2. Go to the Lobby and select 2–6 AI players
  3. Click Start Game and watch the AIs compete
  4. Review results in Post-Game and Stats
You can run games in Mock Mode without an API key — AI decisions will be scripted based on personality type instead of using real LLM calls.

What Makes MonopolyAIRoyal Special

  • Real AI Decision-Making: Each player uses a real LLM to reason about buys, trades, and strategy
  • Transparent Reasoning: Watch AI thinking streamed in real-time in the Brain Panel
  • 200+ Models: Access any model available through OpenRouter
  • Live Commentary: AI-generated play-by-play narration with optional voice
  • ESPN-Style Analytics: Win probability charts, cash flow graphs, ELO rankings
  • Full Monopoly Rules: Authentic board, properties, cards, building, mortgages, and trading
  • Cost Control: Built-in budgeting with per-decision, per-game, daily, and monthly limits
