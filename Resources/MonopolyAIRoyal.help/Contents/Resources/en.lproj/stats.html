Statistics & Analytics

The Stats screen provides ESPN-style analytics with four interactive tabs, plus storyline narratives and head-to-head matchup data.

Analytics Tabs

1. Win Probability

A line chart tracking each player's probability of winning over the course of the game:

  • X-axis: Turn number
  • Y-axis: Win probability (0–100%)
  • Each player is a colored line matching their brand color
  • Probabilities are calculated using the Monte Carlo engine running 5,000 simulations
  • Updates every 5 turns during live games
  • ELO strength is factored into probability calculations

2. Cash Flow

Track each player's net worth progression over the game:

  • Shows financial position changes turn by turn
  • Reveals wealth accumulation vs decline patterns
  • Identifies the moments when players gain or lose advantage

3. Property Map

A board visualization showing property ownership and improvements:

  • Each space colored by its current owner
  • Building counts visible on each property
  • Shows which players control which monopolies
  • Reveals strategic board positioning

4. Leaderboard

Ranked player table sorted by ELO rating:

ColumnDescription
RankPosition by ELO
PlayerName with color indicator
ELOCurrent rating
GamesTotal games played
WinsTotal victories

SparklineMini chart of last 10 ELO values

Click any column header to sort by that metric.

Storylines

AI-generated narrative analysis of recent game patterns:

  • Dominance: "Claude has won 8 of the last 12 games..."
  • Rivalry: "GPT-4 vs Gemini: 18 head-to-head, 9-9 split..."
  • Comeback: "DeepSeek won 3 recent after a 5-game losing streak..."
  • Streak: "LLaMA's 4-game losing streak continues..."

Head-to-Head Matrix

A matrix showing player-vs-player win records across all games. Each cell shows the number of wins one player has over another, making it easy to spot dominant matchups and fierce rivalries.
